# Personal Landing Page

This is a simple personal landing page built using basic HTML and CSS.  
The goal of this project is to practice core frontend concepts and create a clean, minimal personal website.

## Features
- Semantic HTML structure
- External CSS for styling
- Simple and clean layout
- Skills section using bullet lists
- Contact links (email & social)

## Tech Stack
- HTML5
- CSS3

## What I Learned
- How to structure a webpage using semantic tags
- How to link external CSS
- Basic layout using margin and padding
- Styling lists and text
- Keeping code clean and readable

## Future Improvements
- Make the layout responsive
- Add media queries
- Improve design and colors
- Add projects section

## How to Run
Just open `index.html` in your browser.
